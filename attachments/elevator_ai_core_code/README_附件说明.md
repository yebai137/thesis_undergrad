# 毕业设计核心代码附件说明

本附件为《面向电梯场景的 YOLO 双目标检测模型兼容迁移与边缘部署一致性研究》配套的核心源码材料。附件用于展示毕业设计过程中形成的主要代码成果，重点覆盖模型训练评估、模型导出转换、Hi3516DV500 板端应用和实验编排脚本。

## 内容范围

- `src/`：YOLO 训练、验证、预训练基线评估、结果分析和样例可视化脚本。
- `deployment/`：模型权重整理、ONNX 导出、量化准备、ATC 编译辅助和 ONNX 推理检查脚本。
- `board/`：HongOU PI（Hi3516DV500）板端 C 工程源码，包括批量验证、视频文件输入、后处理、OSD、RTSP 输出和主程序入口。
- `tools/server/`：服务器侧数据构建、评估、板端 campaign 编排、视频实验和后处理分析脚本。
- `PROJECT_README.md`、`requirements.txt`、`models_README.md`：项目说明、Python 依赖和模型目录说明。

## 未包含内容

为控制附件体积并避免提交本地环境产物，本附件不包含以下内容：

- 训练数据集、验证集和原始视频；
- 训练输出、实验日志、运行缓存和中间结果；
- PyTorch 权重、ONNX 模型、OM 离线模型等模型文件；
- Hi3516DV500 SDK、CANN、ATC、交叉编译工具链和板端系统库；
- 论文 PDF、论文图片生成缓存、第三方上游源码目录和编译产物。

因此，本附件不是开箱即跑的完整工程发布包，而是毕业设计成果归档用的核心源码包。若需要复现实验，需要另行准备论文中说明的数据集、模型权重、HongOU PI（Hi3516DV500）开发板、AArch64 GNU/glibc 交叉编译工具链，以及 ATC / CANN / SVP_NPU 等部署环境。

## 与论文工作的对应关系

- 训练与评估脚本对应论文中 YOLOv8n 双目标检测模型训练、通用基线评估和训练端性能分析。
- 部署脚本对应 PyTorch 到 ONNX、OM 离线模型的兼容迁移链路。
- 板端 C 工程对应 Hi3516DV500 板端图像验证、视频输入、后处理和分段计时实现。
- 服务器侧 campaign 脚本对应阈值敏感性、NMS 敏感性、后处理消融、视频稳定性和板端验证实验的自动化编排。

## 使用提示

Python 脚本可参考 `requirements.txt` 配置环境；板端工程可参考 `board/README.md` 中的构建与运行方式。由于附件未包含数据、模型和板端 SDK，部分命令需要补齐本地路径和外部依赖后才能执行。
